# Flipkart Sentiment Analysis

## Technology Stack
- Python
- SQL
- Power BI

## Project Steps
1. Loaded and cleaned 189,866 reviews from Flipkart
2. Developed sentiment labels based on star ratings
3. Utilized VADER for text-based sentiment scoring
4. Data stored in SQLite database
5. Developed interactive Power BI dashboard

## Key Insights
- 78% reviews are Positive
- 26% reviews have a mismatch between star rating and review text
- Milton Thermosteel Flask is most reviewed product
